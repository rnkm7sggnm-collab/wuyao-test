# Codex 任务指令模板

请读取 `multi_model_workbench/input.md`，对 ChatGPT、Gemini、DeepSeek 和其他 AI 的回答进行处理。

请完成以下任务：

- 总结每个回答的核心观点
- 对比每个回答的优点和不足
- 去掉重复、空泛、冲突或不可执行的内容
- 保留最有创作价值、最可执行的部分
- 融合成一个新的最终版本
- 输出结构清晰、可直接使用
- 生成 Markdown 文件
- 生成 Word 文档
- 将结果保存到 `multi_model_workbench/output/`

输出要求：

- 最终内容要适合直接用于创作或继续制作
- 保留我的创作方向和偏好
- 不要过度理论化
- 不要堆砌多个模型的原文
- 如果内容方向是故事、视频、AIGC、游戏、工具、App 或交互项目，请按对应媒介整理
- Markdown 文件和 Word 文档的内容应保持一致
