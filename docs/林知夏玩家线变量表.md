# 林知夏玩家线变量表

| 变量名 | 中文名 | 变量含义 | 增加条件 | 减少条件 | 过高会导致什么 | 过低会导致什么 | 影响哪个结局 | 代表性玩家选择 |
|---|---|---|---|---|---|---|---|---|
| `EvidenceScore` | 旧案证据完整度 | 衡量玩家是否掌握旧案死因、动机、身份和共犯结构。 | 收集烧焦账本、汇款单、旧尸检报告、陈伯远录音、出生证明、何青山举报信、陆怀民抱婴照片、林父采访缺页。 | 错过关键证物；证物被陆景川转移或烧毁；只追当前命案不追旧案。 | 可支撑 A 真相公开和 E 隐藏结局。 | 只能模糊怀疑陆景川，容易进入 C 复仇继续或 D 困死废窑。 | A / C / D / E | “周启明可能知道旧案账目。” “死人旁边的东西，不一定是凶器。” “身份和动机，先保住这两样。” |
| `ShenSuspicion` | 沈月娥嫌疑值 | 衡量玩家是否被陆景川栽赃线误导，把沈月娥当成当前凶手。 | 当众质问沈月娥；把釉料粉末直接等同于凶器；误读她的沉默和烧纸灰；最终指认她。 | 私下追问；发现釉料粉末摆放痕迹；找到何青山旧工具；让沈月娥同行进入废窑；证明她保存旧物。 | 进入 B 替罪羊，沈月娥被误认凶手。 | 可洗清沈月娥，打开 A / E。 | A / B / E | “这种粉末只有你懂，你是不是来过账房？” “沈月娥，你知道他说的孩子是谁。” |
| `LuTrust` | 陆景川信任值 | 衡量玩家对陆景川协助、带路和解释的依赖程度。 | 接受陆景川带路；相信他对遗言的解释；把父亲笔记或调查进度告诉他。 | 反问他的知识来源；记录他的脚印和动线；发现工程图缺页；指出他过于熟悉废窑。 | 玩家被带进复仇路线，关键证物可能被转移。 | 过早激怒陆景川，推高 `RevengeTimer`。 | A / C / D / E | “好，那你带我熟悉一下村子。” “你怎么知道这种粉末别人不懂？” |
| `VillageSilence` | 村庄沉默压力 | 衡量青瓷庄集体沉默、回避和阻挠调查的强度。 | 当众公开录音；高压逼问村民；过早揭穿旧案；以外来审判者姿态说话。 | 先建立信任；私下对证；保护证人；承诺完整证据后再公开；让村民看到不是只抓替罪羊。 | 村民集体沉默，难以获得联名反证，E 锁死。 | 可获得村民证词、守祠老人提示或联名反证。 | A / C / E | “大家都听听，二十年前有人撒谎。” “这不是拿来吓人的，是拿来对证的。” |
| `FatherClue` | 父亲线索完整度 | 衡量林知夏是否补完父亲二十年前未能公开的调查。 | 找到父亲旧采访笔记、录音、缺页、暗记；追问陈伯远和何青山死因；保留林父录音用于对证。 | 忽视父亲线；把父亲线索过早交给陆景川；只追当前命案。 | 可进入 E 隐藏结局，让林知夏完成父亲没完成的证词。 | 即使进入 A，也只是普通真相公开，父亲线不完整。 | A / E | “你认识我父亲？” “爸查到的东西，不能再少一次。” |
| `RevengeTimer` | 复仇倒计时 | 衡量陆景川复仇计划推进和情绪失控程度。 | 激怒陆景川；暴露调查进度；让陆景川听到刺激性录音；放任复仇；独自进入废窑。 | 及时阻止；拿到废窑暗道图；用证据让陆景川动摇；理解他但阻止复仇。 | C 复仇继续或 D 困死废窑概率上升。 | 可进入 A / E，但过低不代表自动成功，还需要证据完整。 | A / C / D / E | “我不会拦你。” “你有资格恨他们，但何青山不该只剩下你的复仇。” |
| `SelfDoubt` | 林知夏自我怀疑值 | 衡量玩家误判、被操控和心理动摇的程度。 | 误判沈月娥；证物组合失败；接受错误解释；被陆怀民打击；独自进入废窑。 | 完成正确推理；获得父亲线索；获得沈月娥信任；识破现场布置。 | 关键推理提示减少，D 风险上升。 | 玩家更稳定完成三层最终判断。 | B / D / E | “我只是来写文旅故事。” “死人旁边的东西，不一定是凶器。” |
| `OutsiderTrust` | 外来者信任度 | 衡量村民和非核心人物是否愿意把林知夏当作可信调查者。 | 尊重村民；私下调查；避免一上来指控所有人；理解沈月娥痛苦；保护证人。 | 高压逼问；急于公开；把所有人都当凶手；使用外来审判者姿态。 | 这里的“过高”风险较小，但若只追求讨好村庄，可能错过旧案冲突。 | 锁死部分证词，村庄沉默线难以打开。 | A / C / E | “我只是来写文旅故事，不会碰敏感的事。” “死人需要的不是好结局，是事实。” |

## 变量联动规则

| 联动 | 说明 | 可能结果 |
|---|---|---|
| `ShenSuspicion` 高 + 缺少出生证明 | 玩家最终容易指认沈月娥。 | B 替罪羊 |
| `EvidenceScore` 高 + `ShenSuspicion` 低 | 玩家能证明沈月娥被栽赃，并指向陆景川。 | A 真相公开 |
| `EvidenceScore` 中等 + `LuTrust` 过高 | 玩家知道陆景川可疑，但被他的解释带偏。 | C 复仇继续 |
| `FatherClue` 高 + `VillageSilence` 低 | 父亲线和村庄证词可以互相补强。 | E 隐藏结局 |
| `RevengeTimer` 高 + 无废窑暗道图 | 废窑阶段或最后对质阶段容易失控。 | D 困死废窑 |
| `OutsiderTrust` 高 + `VillageSilence` 低 | 村民更可能提供联名反证或口头证词。 | A / E |
| `SelfDoubt` 高 + 连续错误证物组合 | 林知夏无法稳定完成三层判断。 | B / D |

## 推荐阈值设计

| 阈值方向 | 设计建议 |
|---|---|
| A 真相公开 | `EvidenceScore` 高，`ShenSuspicion` 中低，`RevengeTimer` 未满。 |
| B 替罪羊 | `ShenSuspicion` 高，缺少出生证明或陆怀民抱婴照片。 |
| C 复仇继续 | `EvidenceScore` 中低，或玩家选择放任复仇，或只公开当前命案。 |
| D 困死废窑 | `RevengeTimer` 高，缺少废窑暗道图，且多次激怒陆景川。 |
| E 隐藏结局 | `EvidenceScore` 高，`FatherClue` 高，`ShenSuspicion` 低，`VillageSilence` 低，`RevengeTimer` 未满。 |
