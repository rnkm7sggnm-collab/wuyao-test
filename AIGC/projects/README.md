# Projects

Use this folder for active or archived creative projects.

## Principles

- One project, one folder.
- Name folders clearly, such as `short-film-title/` or `game-prototype-name/`.
- Keep each project focused on a concrete output.
- Prefer usable drafts over abstract planning.
- Preserve the original creative direction before adding structure.

## Suggested Project Layout

- `brief.md` for the core idea, goal, tone, and audience.
- `drafts/` for working versions.
- `assets/` for images, audio, references, or generated media.
- `notes.md` for decisions, questions, and next steps.
