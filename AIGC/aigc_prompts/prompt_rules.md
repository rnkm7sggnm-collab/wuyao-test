# Prompt Rules

## Principles

- Prompts should be specific, compact, and production-ready.
- Preserve the core visual or narrative intent.
- Separate subject, style, composition, lighting, mood, and constraints.
- Avoid unnecessary keyword stuffing.
- Include negative prompts only when they prevent likely problems.

## Working Format

- Goal: what the output should achieve.
- Prompt: the direct generation prompt.
- Settings: model, aspect ratio, duration, or style constraints when relevant.
- Variations: small controlled changes, not random rewrites.
