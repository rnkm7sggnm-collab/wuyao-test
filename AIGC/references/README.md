# References

Use this folder for inspiration, examples, research, and style direction.

## Principles

- Save references with enough context to reuse later.
- Group files by project, medium, or theme.
- Record why a reference matters, not just what it is.
- Keep reference notes short and searchable.
- Do not let references override the user's original creative intent.

## Suggested Notes

- Source or link.
- What to learn from it.
- Tone, style, structure, or technical detail worth reusing.
