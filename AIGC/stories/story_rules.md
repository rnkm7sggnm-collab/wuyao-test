# Story Rules

## Principles

- Start with character, conflict, and tone.
- Keep ideas direct enough to write or pitch immediately.
- Preserve the user's voice, mood, and intended genre.
- Use structure only when it makes the story easier to continue.
- Avoid over-explaining themes; let scenes carry meaning.

## Working Format

- Logline: one sentence.
- Characters: short, memorable descriptions.
- Outline: clear beats, not excessive detail.
- Drafts: write scenes that can be revised directly.
